# 学习休息倒计时

这是一个本地 macOS 原生小应用：

- 启动后先进入学习倒计时，默认 20 分钟。
- 到点后弹出全屏 5 分钟休息倒计时。
- 可以在菜单栏 `设置...` 中修改学习时间，单位是分钟。
- 休息界面会覆盖所有桌面/Space，并显示剩余时间和结束休息时间。
- 休息倒计时过程中，可以点击 `1分钟后休息` 延后休息提醒。
- 休息倒计时过程中，可以点击 `停止休息` 提前结束休息。
- 倒计时结束后不会自动消失，会显示 `继续学习` 和 `停止学习`。
- 点击 `继续学习` 后重新进入 20 分钟学习计时。
- 点击 `停止学习` 后停止计时，应用会留在菜单栏显示 `已停止`。
- Mac 睡眠、合盖或屏幕睡眠时会暂停并关闭倒计时窗口。
- Mac 唤醒后会重新开始一轮完整的 20 分钟学习计时。

## 使用

双击这个应用：

```text
学习休息倒计时.app
```

应用启动后会出现在菜单栏，显示类似 `学习 20:00`。菜单里可以立刻开始休息、重新开始学习、停止学习、打开设置，或退出应用。

应用只会在你点击菜单栏里的 `退出应用` 后结束运行。

当前版本不会自动设置开机启动，也不会写入系统目录；只有你手动启动后才会开始循环。
